Descripción breve del proyecto:
El proyecto UT-Search surge como respuesta a una problemática recurrente en la Universidad Tecnológica de la Costa: la dificultad de los estudiantes para localizar a sus profesores fuera del horario de clase para consultas académicas, asesorías o trámites administrativos. Actualmente, los alumnos deben desplazarse físicamente por las instalaciones, preguntar a diferentes personas o esperar largos períodos sin saber si el docente está disponible o en qué edificio se encuentra.
Esta situación genera pérdida de tiempo, frustración estudiantil y desaprovechamiento de oportunidades de tutoría. UT-Search elimina esta fricción mediante una solución web inteligente que conecta horarios institucionales con visualización geográfica.
