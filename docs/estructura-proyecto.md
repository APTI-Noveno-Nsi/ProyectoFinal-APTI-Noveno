# Estructura Administrativa del Proyecto

## Proyecto
Software House IDGS91

## Rama de trabajo
Feature/Nelson-Frontend

## Responsable
Nelson Ríos Patrón

## Objetivo
Definir una estructura administrativa inicial para organizar la documentación técnica, la configuración del proyecto y los archivos relacionados con la línea base del repositorio.

## Elementos integrados
- Creación de carpeta de documentación.
- Definición de estructura inicial del proyecto.
- Registro de rama individual de trabajo.
- Apoyo a la línea base administrativa del repositorio.

## Importancia
Esta aportación permite mantener un mejor control del repositorio, separar los cambios individuales mediante ramas y facilitar la revisión de avances durante el sprint.
